## Questions
1. :star: Add a method `perimeter`, which takes no arguments and returns the Square's perimeter, to the class below.
    ```java
    public class Square {

        private double side;

        public Square(double s) {
            side = s;
        }

    }
    ```
## Answers
1.
    ```java
    public double perimeter() {
        return 4 * side;
    }
    ```
